# Believer Project

Documentation and parameter change log for the Believer fixed-wing UAV project.

Before starting work, read [`context/startup-prompt.md`](context/startup-prompt.md) — it is the single entry point for getting up to speed on this project. It summarises the project, lists all context files in read order, maps the key documents, and states the standing instructions.

At the start of each session, also check [`context/authorities.md`](context/authorities.md) against the current `.claude/settings.local.json` allow-list. If it lists an authority that isn't yet reflected there, ask the user whether to sync it in before proceeding — don't apply it silently and don't re-ask about gaps the user has already declined.
