# Believer Fixed Wing

Documentation and parameter change log for the Believer fixed-wing project.

## Contents

- `docs/` — manual, ICD, build notes, purchase history, and other documentation
  - `docs/assets/` — diagrams and photos referenced by the docs
- `params/` — parameter change logs and flight controller parameter files
- `context/` — persistent project memory and working directives (see `context/MEMORY.md`)

All project documents are kept as Markdown for clean version control.
