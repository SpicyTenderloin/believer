# Believer Fixed Wing

Documentation and parameter change log for the Believer fixed-wing project.

## Contents

- `docs/` — manual, ICD, build notes, purchase history, and other documentation
  - `docs/assets/` — diagrams and photos referenced by the docs
- `params/` — parameter change logs and flight controller parameter files
- `supporting-documents/` — original proposal, funding application materials, and other source documents kept for reference
  - `supporting-documents/invoices/` — purchase invoices, referenced from `docs/purchase-history.md`
- `context/` — persistent project memory and working directives (see `context/MEMORY.md`)

Project documentation (manual, ICD, notes, logs) is kept as Markdown for clean version control. Supporting documents and invoices are kept in their original format (PDF, etc.) for reference.
